 # Bubble sort java program 
THIS PROGRAM SORTS AN ARRAY OF INTGERS USING THE BUBBLE SORT ALGORITHM.
 ## HOW TO RUN
1. compile:javac Bubble_sort.java
2. run: java Bubble_sort
3. enter the array size and integers when  asked
   ## Example Input:5 5 3 4 2
   ## Output:2 3 4 5 5#java-lab
